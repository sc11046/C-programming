```c
#include <stdio.h>

int main(){
    printf("         ,r'\"7\n");
    printf("r`-_   ,'  ,/\n");    
    printf(" \\. \". L_r'\n");    
    printf("   `~\\/\n");    
    printf("      |\n"); 
    printf("      |\n");                    
    return 0;

}
```

단순하게 출력문을 행 마다 출력한다.

몰랐던 점 : \,  "를 단일로 출력할 때 **\를 하나 더 붙이면 된다.**
